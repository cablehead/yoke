# v0.2.0

First release.

## Highlights

Headless agent harness. JSONL in, JSONL out. Pipes context to any supported LLM provider, runs the tool-call loop, streams events as JSONL to stdout.

- Three providers: Anthropic, OpenAI, Gemini
- Provider-side web search for all three providers (Anthropic server tool, OpenAI search models, Gemini Google Search grounding)
- `--tools` flag for explicit tool selection: `all`, `code`, `web_search`, `none`, or individual tools
- Model discovery: `yoke --provider <name>` lists available models with metadata as JSONL
- Provider discovery: `yoke` with no args lists providers, env vars, and API key dashboard URLs
- Round-trippable JSONL output: context lines (have `role`) feed back as input, observation lines (have `type`) are skipped
- Built-in tools from yoagent: bash, read_file, write_file, edit_file, list_files, search
- Broken pipe handling: exits cleanly when stdout closes
- Cross-platform binaries: darwin-arm64, windows-amd64, linux-arm64, linux-amd64
